Green Space Management Application

Team-based development of a software solution to support an organization managing urban green spaces. The system streamlines planning and maintenance by handling team management, resource allocation, fleet and equipment management, and optimizing irrigation and lighting systems. It also provides statistical insights to assess operational performance.

Key Responsibilities & Skills Used:

  • Agile development following a SCRUM-based methodology, with iterative and incremental (I&I) improvements.
  • Designed UML diagrams covering all software engineering phases, including requirements analysis, system design, and implementation.
  • Developed the application using Java, applying Object-Oriented Programming (OOP) best practices, such as SOLID and GRASP principles.
  • Developed the front-end using JavaFX for the user interface.
  • Implemented object serialization for data persistence across application runs.
  • Strengthened time management, critical thinking, and problem-solving skills in a structured software development environment.

RobDroneGo – Robot Task Execution System

Developed as part of a 5-member team, RobDroneGo is a task execution system prototype designed to manage a fleet of robots and drones. The system consists of three core modules: device management, task request management, and task execution planning. It also supports multiple user roles, including fleet managers, campus managers, administrators, and general users, ensuring efficient task assignment and execution control.

Key Responsibilities & Skills Used:

  • Applied the SCRUM framework, incorporating user stories, sprint backlogs, daily stand-up meetings, and sprint retrospectives.
  • Followed an incremental and iterative development approach to enhance flexibility and continuous improvement.
  • Created UML diagrams at different views and levels of abstraction to model system architecture and workflows.
  • Developed core system functionalities using Node.js for the backend and Angular for the user interface.
  • Integrated 3D visualization features with Three.js, including labeled captions for location identification.
  • Strengthened team collaboration, adaptability to new technologies, and problem-solving skills in a real-world development environment.

Autonomous Robot for ROBOT@FACTORY LITE

Collaborative development of an autonomous robot designed to replicate a potential entry for the ROBOT@FACTORY LITE competition. The project aimed to simulate an industrial setting where the robot navigates a predefined path, detects junctions, and manipulates objects.

Key Responsibilities & Skills Used:

  • Prototype PCB design, though the final implementation used an Arduino Uno.
  • Hardware selection and sensor integration for navigation and object manipulation.
  • Implemented line-following algorithms and intersection detection for autonomous movement.
  • Arduino programming using a state machine pattern for decision-making at junctions.
  • Designed a magnet-based gripping system for handling boxes.
  • Developed problem-solving skills by structuring complex challenges into manageable tasks.
  • Gained experience in team collaboration, adaptability, and hands-on engineering experience in a project-based environment.
